# Pandas CSV Reader & Basic Analysis Project

## 📌 Project Overview
This project demonstrates how to use **Python Pandas** to read, clean, analyze, and extract meaningful insights from a real-world sales dataset.

The goal of this project is to apply core Pandas concepts and generate business-oriented outputs.

---

## 🎯 Objectives
- Read CSV data into a DataFrame
- Inspect data using head, tail, info, and data types
- Handle missing values
- Convert date columns into proper datetime format
- Compute summary statistics (mean, median, min, max, count)
- Filter and slice data
- Generate real-life insights from the dataset
- Save processed data into CSV files

---

## 🛠️ Tools Used
- Python
- Pandas

---

## 📊 Key Analysis Performed

### 🔹 Data Inspection
- First and last rows
- Data types and structure
- Null value analysis

### 🔹 Data Cleaning
- Filled missing values in Postal Code
- Converted date columns using `pd.to_datetime()`

### 🔹 Summary Statistics
- Mean
- Median
- Minimum
- Maximum
- Count

### 🔹 Data Filtering & Selection
- High sales orders (Sales > 500)
- Furniture category analysis
- West region analysis

### 🔹 Data Slicing
- Row and column slicing using `.iloc`

### 🔹 Feature Engineering
- Extracted Month from Order Date

### 🔹 Business Insights
- Monthly sales trends
- Region-wise sales performance
- Category-wise sales distribution
- Top 10 products by sales
- Top 10 states by sales
